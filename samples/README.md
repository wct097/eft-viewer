# Sample EFT Files

This directory is a placeholder for sample Electronic Fingerprint Transmission (EFT) files.

## Obtaining Test Files

NIST provides fingerprint databases that can be used for testing:

- [NIST Special Database 300](https://www.nist.gov/itl/iad/image-group/nist-special-database-300) - Uncompressed fingerprint images from cards
- [NIST Special Database 302](https://www.nist.gov/itl/iad/image-group/nist-special-database-302) - Nail-to-nail fingerprint images
- [ANSI/NIST-ITL Standard References](https://www.nist.gov/itl/iad/image-group/ansinist-itl-standard-references) - Schema samples and reference files

See the project README for more information on the EFT/ANSI-NIST-ITL format.
